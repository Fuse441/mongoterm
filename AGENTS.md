# AGENTS.md

This repository's single source of truth for architecture, conventions, and
known gotchas lives in **[`CLAUDE.md`](./CLAUDE.md)** — read it in full
before making any change, and update the matching section in the same
change whenever you touch something it documents (see its own
"Keeping this file in sync" table at the top).

This file exists because `AGENTS.md` is the filename several AI coding
tools look for by convention. If your tool reads `AGENTS.md` specifically,
treat everything in `CLAUDE.md` as if it were written here — this is not a
separate or lighter set of rules, just a pointer so every tool converges on
one document instead of drifting copies.
